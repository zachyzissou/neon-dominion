# Visual language

## Palette rule
Charcoal/black base + **one phosphor accent**.

Example accents:
- Green `#00FF66`
- Amber `#FFB400`
- Teal `#9EE6E6`

## Typography rule
- Prefer ALL CAPS labels, increased tracking.
- Monospaced alignment for data tables.

Candidate fonts (prototype): OCR-A, Eurostile/Microgramma-like, VT323, Share Tech Mono.

## Composition
- Modular frames + strict grids.
- Occasional radial telemetry.
- Hardware cues: bezels, screws, stamped IDs.

## Motion
- Flicker/type-in/burst updates.
- Micro-pulses to imply “alive systems.”
- Avoid modern silky transitions as the default.

## Copy tone
Instrumentation language:
- `POWER ROUTING: STABLE`
- `SECURITY STATE: ELEVATED`
- `MORALE INDEX: 0.62`
